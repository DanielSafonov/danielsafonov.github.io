Learn Spring Boot

12 May 2019 | spring java

Материалы для изучения Spring Framework.

Подход к изучению

Spring Framework традиционно считается одним из самых сложных и при этом высокооплачиваемых. Однако, существующие подходы к его изучению крайне неэффективны. <..> Целенаправленное обучение - это быстрый, но ОЧЕНЬ тяжёлый способ. Он заключается в том, что человек продвигается по книге (или курсу), а потом пытается применить это на практике и как-то запомнить. И если с Java SE такой подход ещё кое-как работает, то со спрингом всё глухо и туго. Даже в самых лучших книгах зачастую не объясняется, где конкретно применять те или иные особенности, приходится догонять это самому. Но самое обидное тут — это забывание информации, добытой таким тяжёлым трудом.

habr: Изучение Spring Framework 100% через практику.

По своему опыту могу сказать, что неплохой вариант - начать поверхностное изучение Spring Boot на практике, разрабатывая какой-то pet project по руководству для новичков, не погружаясь в детали и внутреннее устройство Spring`а (например, с видеокурса LetsCode, как я и поступил).

После этого следует перейти к более глубокому изучению отдельных элементов фреймворка для решения конкретных задач, постепенно изучая его устройство.

Интернет-ресурсы:

  • Spring.io/guides - официальный сборник туториалов;
  • Baeldung - сборник туториалов;
  • Mkyong - блог Java разработчика;
  • Spring Boot Docs - официальная документация;
  • Spring-project.ru [ru] - частичный перевод spring.io;
  • o7planning [ru] - сборник туториалов;
  • DZone - сборник туториалов (не только spring);
  • LetsCode - [video] видеокурс на YouTube, в котором автор рассказывает о Spring Boot на примере пошаговой разработки клона твиттера;
  • Spring Petclinic [code] - официальное приложение-пример от создателей Spring Framework + еще проекты.

Книги:

  • Spring in Action 5th Edition - Craig Walls
    (Перевод - “Spring в действии”)
  • Pro Spring 5: An In-Depth Guide to the Spring Framework and Its Tools 5th ed. Edition - Iuliana Cosmina, Rob Harrop, Chris Schaefer, Clarence Ho
    (Перевод - “Spring 5 для профессионалов”)
  • Cloud Native Java: Designing Resilient Systems with Spring Boot, Spring Cloud, and Cloud Foundry - by Josh Long, Kenny Bastani
    (Перевод - “Java в облаке. Spring Boot, Spring Cloud, Cloud Foundry”)
