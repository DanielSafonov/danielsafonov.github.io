Семейства шрифтов

  • serif — шрифты с засечками на концах;
  • sans-serif — шрифты без засечек;
  • cursive — шрифты курсивного начертания;
  • fantasy — декоративные шрифты;
  • monospace — моноширинные шрифты (с буквами одинаковой ширины).

Безопасные шрифты

На разных устройствах предустановлены разные наборы шрифтов. Существую так называемые “безопасные” шрифты, с очень высокой долей вероятности установленные на всех устройствах. Ниже представлены некоторые из них.

# font-face family
1 Arial sans-serif
2 Arial Black sans-serif
3 Verdana sans-serif
4 Tahoma sans-serif
5 Times serif
6 Courier New monospace
7 Georgia serif
8 Helvetica sans-serif
9 Palatino serif
10 Bookman serif

Помимо этого, использование стандартных (безопасных) шрифтов позволит сократить время загрузки страницы.

Подключение шрифтов

Необходимо обязательно указывать generic шрифт.

  p {
    font-family: 'My Awersome Custom Font', sans-serif;
  }

  code {
    font-family: 'My Code Font', monospace;
  }
