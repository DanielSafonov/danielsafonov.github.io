Color Schemes

04 Oct 2018 | design color

Цвет важен.

Генераторы цветовых схем:

  1. color.adobe - классика
  2. color.hailpixel - построение цветовой схемы движением указателя
  3. mycolor.space, colors.co, colorhunt, canva
  4. canva - генерация цветовой схемы по загруженной картинке
  5. colormind - использует в алгоритмах подбора цвета Deep Learning
  6. colorsafe - подбор цветов для читабельного текста (соотношение контрастности цвета текста и фона по WCAG Guidelines)
