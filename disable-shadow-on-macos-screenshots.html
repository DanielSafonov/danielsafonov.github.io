Disable shadow on Mac OS screenshots

04 May 2019 | macos

Достаточно выполнить всего одну команду в консоли:

$ defaults write com.apple.screencapture disable-shadow -bool true

…и перезапустить System UI Server для примнения изменений в силу:

$ killall SystemUIServer

Для возврата тени:

$ defaults write com.apple.screencapture disable-shadow -bool false
$ killall SystemUIServer
