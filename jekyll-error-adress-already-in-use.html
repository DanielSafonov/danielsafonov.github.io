Jekyll address already in use

15 May 2019 | jekyll 
jekyll 3.8.5 | Error:  Address already in use - bind(2) for 127.0.0.1:4000

Данная ошибка возникает при попытке запуска Jekyll-сервера (jekyll serve) в случае наличия уже запущенного аналогичного процесса.

Для решения проблемы достаточно завершить этот процесс.

#Команда выводит список всех активных процессов jekyll
$ ps aux | grep jekyll 
#Команда завершает процесс с ID = PID (вторая колонка после выполнения предыдущей команды)
$ kill -9 PID
