Script to publish jekyll

14 Sep 2019 | jekyll

Скрипт собирает, коммитит и пушит сначала в приватный репозиторий с исходниками сайта, а затем в открытый репозиторий-хостинг (github pages).

$ touch upload.sh #Создать файл
$ nano upload.sh #Перейти к редактированию файла

jekyll build
git add *
git commit -m "Update"
git push origin master
cd _site
git add *
git commit -m "Update"
git push origin master
echo -e "\n\n\n\e[32mBlog was uploaded successfully\033[0m\n\n\n"

$ chmod +x upload.sh #Сделать скрипт исполняемым
$ ./upload.sh #Выполнить скрипт
